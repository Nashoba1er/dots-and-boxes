# Dots and boxes tutorial

## Menu principal

le menu principal permet de choisir à quel mode de jeux vous souhaitez jouer.
Vous pouvez également choisir les couleurs des joueurs et du fond d'écran en cliquant sur le menu couleurs
Vous pouvez revenir à ce menu à tout moment en cliquant sur la touche m.

## Menu couleurs

Dans ce menu vous pouvez choisir quelle couleur vous voulez pour le fond, et pour les joueurs.

## Menu dimension

Ce menu apparaît quand vous avez choisi votre mode de jeu.
Vous pouvez définir la taille du plateau

## Menu de jeu

Cet interface vous permet de jouer au mode de jeu que vous avez sélectionné.
Si vous avez choisi joueur contre joueur,
- le clic gauche de la souris est le bouton du joueur 1
- le clic droit est celui du joueur 2.

Vous pouvez appuyer sur r à tout moment pour recommencer la partie.
